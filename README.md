# infrared-agents

Public AI agent skills and plugins for the [Infrared](https://infrared.city) urban simulation platform.

## Focus Areas

- **Urban Planning & Design** — AI-assisted urban analysis, site evaluation, zoning
- **Climate Simulation & Analysis** — UHI, wind comfort, solar access, flood risk, energy
- **SDK Integration** — Agent workflows powered by the Infrared SDK/API

## Status

Research phase — surveying the landscape of existing urban AI agent tools and defining our approach.
